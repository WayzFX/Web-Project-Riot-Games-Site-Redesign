# Riot Games Website - Web Integration Project

Ce projet consiste en l'intégration web d'une maquette graphique dédiée à Riot Games. L'objectif était de transformer une conception visuelle (Figma) en une interface web interactive, moderne et responsive, en utilisant les bonnes pratiques HTML5 et CSS3.

**Lien de la maquette Figma entièrement créée par moi-même :** https://www.figma.com/design/gtUSkqfBXOsLDKXDMKZlJO/SAE1_RAHMANI_NASSER_Touboulic_Victor?node-id=0-1&t=Fh0uxgQDaX8pb21U-1
---

## Fonctionnalités Clés

* **Intégration fidèle :** Conversion d'une maquette Figma en structure web sémantique.
* **Responsive Design :** Mise en page flexible utilisant `Flexbox` et `CSS Grid` pour s'adapter à différents écrans.
* **Interactivité avancée :**
    * **Cartes interactives :** Effets de survol (flip cards) créés avec les transitions CSS.
    * **Carrousels :** Navigation horizontale fluide pour la section des jeux.
    * **Animations :** Transitions douces au survol des éléments et comportement de défilement fluide (`scroll-behavior: smooth`).
* **Structure Sémantique :** Utilisation propre des balises HTML (`header`, `main`, `footer`, `section`) pour une meilleure accessibilité.

---

## Stack Technique

* **Langages :** HTML5, CSS3.
* **Outils :** Figma, VS Code.
